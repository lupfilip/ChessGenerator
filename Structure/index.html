Chess position generator

FEN:

This Generator:

- Generates position on random algorithm.

- Generates pieces on their half of the board.

- One pawn per color is generated on each row.

- There is no illegal position.

- Can generate finished game states (stalemate/checkmate).

- Cannot make moves.

- Checkbox for manual piece selection.

- Manual piece selection is limited (to a chess set and promotions).


A small project where it makes positions.
